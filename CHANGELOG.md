# Changelog

## 2.1.0 - 2026-04-19

- **type**: MINOR
- **title**: IA再設計・相手詳細ページ新設・取引フィルター強化

### 追加

- 相手詳細ページを新設し、取引履歴・残高・共有リンクをまとめて確認できるようになりました
- すべての取引を横断的に確認できる「すべての取引」ページを新設しました
- 統計ページを新設しました
- 検索・絞り込みシートを追加し、条件付きフィルタリングに対応しました
- 共有リンクに「リンクを開く」ボタンを追加しました

### 改善

- ホームを相手一覧ページに変更し、残高・最終取引日を一目で確認できるようになりました
- ボトムバーを相手・すべての取引・統計・メニューの4タブ構成に変更しました
- 取引カードに相手との累積残高を表示するようになりました
- 残高カード・共有リンクカード・取引カードのデザインをエメラルドテーマに刷新しました
- ページに応じた取引作成ボタンの表示を最適化しました
- 取引金額のソートを絶対値ベースに変更しました
- 相手の名前変更・アーカイブ・削除を相手詳細ページからインラインで操作できるようになりました
- 精算機能を廃止し、UI をシンプルにしました

## 2.0.0 - 2026-04-08

- **type**: MAJOR
- **title**: UI全面刷新・URL共有・ヘルプ追加

### 追加

- 取引内容をリンクで共有できる URL 共有機能を追加しました
- ヘルプページを追加しました
- N キーで取引作成モーダルを開くキーボードショートカットを追加しました

### 改善

- 取引の作成・編集モーダルを刷新し、アーカイブ機能を追加しました
- ホーム画面をタブなし単一スクロールページに刷新しました
- 取引一覧・相手カードをカード表示に変更し、フィルタ機能を追加しました
- 取引モーダルの相手選択 UI をボタングリッドに変更しました
- テーマカラーをエメラルドグリーンに変更しました
- 統計ページ・取引ページの UI を改善しました
- ボトムバーをメニューボタンに変更し、設定・ヘルプへの導線を追加しました
- 設定ページにバージョン表示を追加しました
